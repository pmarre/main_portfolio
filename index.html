Sandy's Pet Shop

For this project, I was prompted to created a multipage website that connected to a MySQL Database. Some of the features include forms submitted with AJAX, contact emails sent using PHPMailer, and an admin page with a table of all clients who have submitted a grooming form. The admin can add, update, and delete clients.

More Goals:

  • Fully functioning website
  • Location API using Google Maps
  • Responsive Layout

Admin Page

The admin has the ability to view, update, delete, and add clients to table. This is achieved using PHP, AJAX, SQL, and a Bootstrap Module.
